We are developping 3 open-source open source softwares

These softwares were initiated based on the specific needs of Social Innovations Crossroad. They enable to collect, aggregate, consolidate and visualize information on social innovation projects.

We also invest in digital investment. These softwares are open-source in an operational way. They are starting to be adapted to other needs of actors working the public interest. the 3 softwares can be used separately.

  • Open scraper collects projects information directly on the websites.

  • Solidata aggregates and consolidates projects information.

  • Apiviz is where to visualise information thanks to a search engine, a map and data visualisation tools.

Open Scraper

To collect data on websites

Features

  • Collect data published on websites
  • Turn disparate data into homogenous ones
  • Open data and control the degree of their sharing (from private data to open data)
Check on Github

Solidata

To aggregate and consolidate data

Features

  • Aggregate data coming from diverse files and sources
  • Enrich data thanks to external open data : geolocalisation, administrative data, …
  • Open data and control the degree of their sharing (from private data to open data)
Check on Github

Apiviz

To visualise data flow

Features

  • Look for precise data thanks to a search engine
  • Visualise aggregated data in different ways : map, data visualisation, etc.
  • Make its own favorite list
Check on Github
